A structured, bottom-up learning repository covering Python & CS fundamentals, Machine Learning and AI (from math foundations to implementation), and Operations Research (LP, MIP, graph optimization, dynamic programming).

Everything is implemented from scratch — the goal is genuine understanding, not library calls. This is memory practice from my Master's degree in Data Science, revisited deliberately because repetition is key for long term memorization. In a world of AI shortcuts and pressure to deploy fast, this repository is about taking your time, repeating fundamentals, and building deep understanding through practice.

Built as part of a self-directed learning itinerary toward expertise in ML/AI and Optimization, with applications in energy systems.
